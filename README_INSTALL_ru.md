
# ИНСТРУКЦИЯ ПО УСТАНОВКЕ Meth0d

## ТРЕБОВАНИЯ
**Node.js** должен быть установлен: [https://nodejs.org/](https://nodejs.org/).

## ПОДГОТОВКА
1.  Запустите `removetxt.bat`.
    *Это превратит текстовые файлы скриптов в исполняемые.*

## ВАРИАНТ А: СОЗДАНИЕ УСТАНОВЩИКА (Рекомендуется)
Этот способ создаст настоящий `.exe` файл для установки в систему.

1.  Запустите `generate_system_exe.bat`.
    *Процесс может занять несколько минут. Он скачает зависимости и соберет программу.*
2.  После завершения откройте новую папку **`Install_files`**.
3.  Запустите установочный файл (например, `Meth0d Setup 2.1.0.exe`).
4.  Приложение установлено!

## ВАРИАНТ Б: БЫСТРЫЙ ЛОКАЛЬНЫЙ ЗАПУСК
Без установки в систему.

1.  Запустите `setup_app.bat`.
2.  Запустите `run_meth0d.bat`.
    *Не закрывайте черное окно консоли.*

## БЭКАПЫ
Ваши данные хранятся локально.
*   **Архив -> Скачать Резервную Копию**: Сохранить данные в файл.
*   **Архив -> Загрузить Бэкап**: Восстановить данные.
