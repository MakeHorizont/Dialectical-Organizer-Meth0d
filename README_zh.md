**Meth0d** 是一个专门用于辩证唯物主义分析的渐进式网络应用程序 (PWA)。它通过严格的4阶段协议将混乱的现实转化为结构化的战略。这不仅仅是一个待办事项清单 —— 它是思想的武器。

## 🌀 哲学：对数螺线

该应用程序建立在**质的飞跃**这一概念之上。在辩证法中，发展是螺旋式的，而不是圆形的。螺旋的每一圈似乎都在重复过去，但处于更高、更复杂的水平。

> *"改变后，我以更好的姿态重生。"*

我们不仅仅是解决问题；我们“扬弃”（Aufheben）它们 —— 解决矛盾，将系统提升到一个新的存在状态。

## ☭ 核心功能

*   **协议向导：** 一个引导式向导，带您完成分析的4个阶段：
    1.  **分解（向量）：** 历史、阶级、系统和意识形态分析。
    2.  **综合：** 识别推动现象的主要矛盾。
    3.  **战略：** 识别系统漏洞和行动机会。
    4.  **审计：** 自我批评和风险评估。
*   **语义主题：** 适应您的思维模式。包括“社会主义”（红色）、“中性”（企业/结构）、“野兽派”（高对比度）等。
*   **离线优先：** 构建为具有本地加密的 PWA。除非您共享，否则您的数据永远不会离开您的设备。
*   **教育：** 内置交互式教程，解释辩证唯物主义和历史唯物主义的基础。

## 🛠 安装与使用

**[ 🇨🇳 安装指南 ](README_INSTALL_zh.md)**

### Web / PWA (开发)

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 桌面 (Windows/Mac) - 快速开始

我们提供批处理脚本以进行快速本地部署，无需立即使用 Electron：

1.  运行 `removetxt.bat` 以准备脚本。
2.  运行 `setup_app.bat` 以构建应用程序。
3.  运行 `run_meth0d.bat` 以启动本地服务器。

### 桌面 (Electron)

要构建原生可执行文件 (`.exe`, `.dmg`, `.AppImage`)：

```bash
# 为当前操作系统构建
npm run electron:build
```

输出将位于 `release` 文件夹中。
