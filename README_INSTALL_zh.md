
# Meth0d 安装指南

## 要求
必须安装 **Node.js**：[https://nodejs.org/](https://nodejs.org/)。

## 准备工作
1.  运行 `removetxt.bat`。
    *这将把文本文件转换为可执行脚本。*

## 选项 A：生成安装程序（推荐）
这将创建一个真正的安装文件（.exe 等）。

1.  运行 `generate_system_exe.bat`。
    *此过程需要几分钟时间。*
2.  完成后，打开 **`Install_files`** 文件夹。
3.  运行适合您系统的安装程序（例如 `Meth0d Setup 2.1.0.exe`）。
4.  Meth0d 现已安装！

## 选项 B：本地服务器
无需安装即可运行。

1.  运行 `setup_app.bat`。
2.  运行 `run_meth0d.bat`。
    *保持控制台窗口打开。*

## 数据备份
您的数据存储在本地。
*   **档案馆 (Archive) -> 下载备份**：保存数据。
*   **档案馆 (Archive) -> 导入备份**：恢复数据。
