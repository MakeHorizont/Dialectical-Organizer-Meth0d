
# Meth0d स्थापना निर्देश

## आवश्यकताएँ
**Node.js** स्थापित होना चाहिए: [https://nodejs.org/](https://nodejs.org/)।

## तैयारी
1.  `removetxt.bat` चलाएं।
    *यह टेक्स्ट फ़ाइलों को निष्पादन योग्य स्क्रिप्ट में बदल देगा।*

## विकल्प A: सिस्टम इंस्टॉलर (अनुशंसित)
यह सिस्टम में स्थापित करने के लिए एक वास्तविक प्रोग्राम फ़ाइल (.exe) बनाएगा।

1.  `generate_system_exe.bat` चलाएं।
    *इस प्रक्रिया में समय लग सकता है।*
2.  पूरा होने पर, **`Install_files`** फ़ोल्डर खोलें।
3.  इंस्टॉलर फ़ाइल चलाएं (जैसे `Meth0d Setup 2.1.0.exe`)।
4.  Meth0d अब स्थापित है!

## विकल्प B: स्थानीय सर्वर
बिना इंस्टॉल किए चलाने के लिए।

1.  `setup_app.bat` चलाएं।
2.  `run_meth0d.bat` चलाएं।
    *कंसोल विंडो खुली रखें।*

## डेटा बैकअप
आपका डेटा स्थानीय है।
*   **अभिलेखागार (Archive) -> बैकअप डाउनलोड करें**: डेटा सहेजें।
*   **अभिलेखागार (Archive) -> बैकअप आयात करें**: डेटा पुनर्स्थापित करें।
