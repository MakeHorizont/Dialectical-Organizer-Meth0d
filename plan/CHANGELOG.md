
# Changelog

Все существенные изменения проекта будут зафиксированы в этом файле.

## [2.2.0] - 2025-11-28
### Added
- **Поиск и Фильтрация:** На главном экране (Dashboard) появилась панель поиска и переключатели статуса (В работе / Готовые).
- **Markdown Export:** В просмотре анализа добавлена кнопка "Скачать .MD" для интеграции с Obsidian, Notion и GitHub.
- **Печать / PDF:** Добавлена функция печати "Стратегического Отчета". Специальные стили скрывают интерфейс и адаптируют контент для бумаги (шрифт с засечками, черно-белый режим).
- **Drag-and-Drop Импорт:** В разделе Архив теперь можно восстановить данные, просто перетащив JSON файл в специальную зону.
- **UI Tweaks:** Кнопки навигации в Визарде стали значительно крупнее.
- **UX Tweaks:** Нижняя панель навигации теперь скрывается при создании или редактировании анализа, чтобы не отвлекать пользователя.

## [2.1.1] - 2025-11-28
### Added
- Добавлена кнопка "Редактировать" (карандаш) в панели контекста Визарда. Позволяет вернуться к любому предыдущему шагу Фазы 1.
- Добавлена визуальная индикация при копировании текста через кнопку "Поделиться".
- Создана структура документации в папке `plan/`.

### Changed
- Увеличен размер кнопок навигации (Назад/Далее) для удобства использования.
- Оптимизирован механизм генерации мотивирующих фраз (Feedback) в Визарде: теперь текст не "мерцает" при вводе.
- Обновлен механизм "Поделиться": добавлен надежный фоллбэк на буфер обмена для Desktop-версий.

## [2.1.0] - 2025-11-26
### Added
- Полноценная поддержка сборки для Windows через Electron.
- Скрипты автоматической сборки (`generate_system_exe.bat`).
- Локализация (RU, EN, ES, ZH, HI).
- Темизация (8 цветовых тем).

### Fixed
- Исправлены проблемы с путями в Vite для Electron (переход на хеш-роутер и относительные пути).
- Убраны внешние CDN зависимости для полной автономности.
