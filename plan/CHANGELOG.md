
# Changelog

Все существенные изменения проекта будут зафиксированы в этом файле.

## [2.6.0] - 2025-11-28
### Major Changes
- **Системные Связи (Systemic Connections):** Добавлена возможность связывать разные анализы между собой (двусторонняя связь). Это позволяет выстраивать цепочки причинно-следственных связей между разными объектами исследования.
- **Интерфейс Связей:** В карточке анализа добавлен блок отображения связанных объектов с возможностью быстрого перехода.

## [2.5.0] - 2025-11-28
### Major Changes
- **Дневник Практики (Praxis Log):** В карточку завершенного анализа добавлен функционал ведения журнала. Теперь можно фиксировать ход выполнения стратегии, новые события и результаты. Это превращает Meth0d из генератора отчетов в инструмент сопровождения проектов.

## [2.4.0] - 2025-11-28
### Major Changes
- **Система Тегов (Labels):** Внедрена возможность добавлять теги к анализу на этапе создания. Теги отображаются на Дашборде и в отчетах.
- **Визуальная Матрица:** В режиме просмотра отчета (AnalysisView) ответы Фазы 1 теперь организованы в сетку 2x2 (Квадрант), а не вертикальный список.

### Changed
- **UI:** Убрано агрессивное цветовое выделение активного языка на экране выбора. Теперь интерфейс выбора языка нейтрален.

## [2.3.0] - 2025-11-28
### Major Changes
- **Структуризация (Матрица):** Визуализация 4 модулей анализа (Фаза 1) теперь представлена в виде наглядной сетки 2x2, а не простого списка.
- **Система Тегов:** Добавлена возможность присваивать категории (теги) для каждого анализа (например: Работа, Личное, Политика). Теги отображаются на дашборде и в отчетах.

### Added
- **Сценарии (Шаблоны):** Добавлен блок "Типовые сценарии" на стартовый экран. Позволяет в один клик заполнить название и тезис для типичных проблем (Конфликт, Застой).
- **Экспорт JSON (Single):** Добавлена возможность выгрузить один конкретный анализ в JSON для обмена с товарищами.

## [2.2.0] - 2025-11-28
### Added
- **Поиск и Фильтрация:** На главном экране (Dashboard) появилась панель поиска и переключатели статуса (В работе / Готовые).
- **Markdown Export:** В просмотре анализа добавлена кнопка "Скачать .MD" для интеграции с Obsidian, Notion и GitHub.
- **Печать / PDF:** Добавлена функция печати "Стратегического Отчета". Специальные стили скрывают интерфейс и адаптируют контент для бумаги (шрифт с засечками, черно-белый режим).
- **Drag-and-Drop Импорт:** В разделе Архив теперь можно восстановить данные, просто перетащив JSON файл в специальную зону.
- **UI Tweaks:** Кнопки навигации в Визарде стали значительно крупнее.
- **UX Tweaks:** Нижняя панель навигации теперь скрывается при создании или редактировании анализа, чтобы не отвлекать пользователя.

## [2.1.1] - 2025-11-28
### Added
- Добавлена кнопка "Редактировать" (карандаш) в панели контекста Визарда. Позволяет вернуться к любому предыдущему шагу Фазы 1.
- Добавлена визуальная индикация при копировании текста через кнопку "Поделиться".
- Создана структура документации в папке `plan/`.

### Changed
- Увеличен размер кнопок навигации (Назад/Далее) для удобства использования.
- Оптимизирован механизм генерации мотивирующих фраз (Feedback) в Визарде: теперь текст не "мерцает" при вводе.
- Обновлен механизм "Поделиться": добавлен надежный фоллбэк на буфер обмена для Desktop-версий.

## [2.1.0] - 2025-11-26
### Added
- Полноценная поддержка сборки для Windows через Electron.
- Скрипты автоматической сборки (`generate_system_exe.bat`).
- Локализация (RU, EN, ES, ZH, HI).
- Темизация (8 цветовых тем).

### Fixed
- Исправлены проблемы с путями в Vite для Electron (переход на хеш-роутер и относительные пути).
- Убраны внешние CDN зависимости для полной автономности.
