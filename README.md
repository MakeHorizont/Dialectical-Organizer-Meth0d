### Web / PWA (Development)

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Desktop (Windows/Mac) - Quick Start

We provide batch scripts for quick local deployment without needing Electron immediately:

1.  Run `removetxt.bat` to prepare the scripts.
2.  Run `setup_app.bat` to build the application.
3.  Run `run_meth0d.bat` to launch the local server.

### Desktop (Electron)

To build a native executable (`.exe`, `.dmg`, `.AppImage`):

```bash
# Build for your current OS
npm run electron:build
```

The output will be in the `release` folder.
