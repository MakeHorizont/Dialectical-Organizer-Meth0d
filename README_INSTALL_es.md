
# INSTRUCCIONES DE INSTALACIÓN DE Meth0d

## REQUISITOS
**Node.js** debe estar instalado: [https://nodejs.org/](https://nodejs.org/).

## PREPARACIÓN
1.  Ejecute `removetxt.bat`.
    *Esto convierte los archivos de texto en scripts ejecutables.*

## OPCIÓN A: INSTALADOR NATIVO (Recomendado)
Crea un programa real (.exe, etc.) para instalar en el sistema.

1.  Ejecute `generate_system_exe.bat`.
    *Este proceso toma tiempo. Construye la aplicación.*
2.  Al finalizar, abra la carpeta **`Install_files`**.
3.  Ejecute el archivo instalador (ej. `Meth0d Setup 2.1.0.exe`).
4.  ¡Meth0d está instalado!

## OPCIÓN B: SERVIDOR LOCAL
Para usar sin instalar.

1.  Ejecute `setup_app.bat`.
2.  Ejecute `run_meth0d.bat`.
    *Mantenga la ventana de la consola abierta.*

## COPIAS DE SEGURIDAD
Sus datos son locales.
*   **Archivos -> Descargar Copia de Seguridad**: Guardar datos.
*   **Archivos -> Importar Respaldo**: Restaurar datos.
